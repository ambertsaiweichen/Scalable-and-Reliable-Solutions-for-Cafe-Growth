## 🔎 Overview
Designed an enterprise-grade AWS cloud architecture for a growing café business expecting significant traffic growth from media exposure.<br/>

The solution focuses on:

▪️ High Availability<br/>
▪️ Scalability<br/>
▪️ Security<br/>
▪️ Cost Optimisation<br/>
▪️ Operational Resilience<br/>

## 📋 Business Scenario
A café business expects a large increase in website traffic following a television appearance.

Key requirements:

▪️ Support 10,000+ users<br/>
▪️ Minimise downtime<br/>
▪️ Protect customer data<br/>
▪️ Maintain performance during traffic spikes<br/>
▪️ Keep operational costs manageable<br/>

## 💡 Architecture Highlights
### Reliability
- Multi-Availability Zone deployment
- Auto Scaling for traffic spikes
- Load balancing across application servers
- Backup database architecture
### Security
- Identity and Access Management (IAM)
- Bastion Host for administrative access
- AWS WAF for web application protection
- Private/Public subnet separation
### Performance
- CloudFront Content Delivery Network
- Horizontal scaling strategy
- Traffic distribution through ALB
### Cost Optimisation
- Right-sized EC2 instances
- Auto Scaling resource management
- Cost analysis using AWS Pricing Calculator
